Portfolio Example

Project: 3D Audio Processing Framework

This project demonstrates a modular Python-based framework for 3D audio rendering, focusing on spatial sound simulation and real-time processing. Key features include:

  • Binaural audio rendering using head-related transfer functions (HRTFs).
  • Real-time 3D audio spatialization with user-defined sound sources.
  • Integration with TDD using pytest for robust algorithm testing.
  • Visualization tools for sound source positioning and audio waveforms.
