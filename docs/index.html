Portfolio Example

Project: Real-Time Spatial Audio Processing Framework

A production-ready Python monorepo implementing modular signal processing architecture with spatial audio capabilities. Built following strict TDD principles with comprehensive test coverage.

Architecture Highlights

  • Modular Design: Separate packages for STFT processing, spatial audio, buffer management, and activation patterns
  • Binaural Rendering: HRTF-based 3D audio with quaternion-based head orientation tracking
  • Real-Time GUI Demo: Tkinter-based interface with live azimuth/elevation control and per-channel gain management
  • Activator Pattern: Abstract base class with file-based and real-time callback implementations
  • TDD Throughout: 100+ tests across all modules using pytest with YAML-based parametrization

Technical Stack

  • Python 3.12+ with type hints and modern tooling (uv, pre-commit hooks)
  • NumPy for signal processing, PyAudio for real-time I/O
  • STFT with perfect reconstruction, configurable overlap ratios
  • CI/CD with GitHub Actions (automated testing on all PRs)
